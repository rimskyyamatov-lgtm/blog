# プログラミング言語の利点
`C,C++,C#,Fortran,Go,Haskell,Javascript,Java,VB.net,Ruby,Rust,Python`

### 今回はこの12の言語の利点や用途をまとめる。
---

##  低レイヤ・高速系

### ■ C

* 圧倒的に軽量（ランタイムほぼ無し）
* OS・組み込みで最強クラスの実績
* メモリを完全に自分で制御できる
* コンパイラが超成熟（どの環境でも動く）

「ハード寄り・制御したいなら最強」

---

### ■ C++

* C並みの速度 + 高レベル抽象（両取り）
* テンプレートでゼロコスト抽象化
* ゲーム・リアルタイム処理で強い
* STLでそこそこ便利

「速度を落とさずに高機能を書ける」

---

### ■ Rust

* メモリ安全（GCなしで安全）
* データ競合をコンパイル時に防ぐ
* C/C++並みに速い
* 並行処理が強い

「安全なC++みたいな存在」

---

### ■ Fortran

* 数値計算が異常に速い（最適化が強い）
* 配列・行列処理がネイティブに強い
* 科学計算ライブラリが豊富（歴史）

「ガチ数値計算なら未だに最強クラス」

---

## バランス・システム系

### ■ Go

* シンプルで覚えやすい
* goroutineで並行処理が超簡単
* ビルドが速い・単一バイナリ
* サーバー開発に最適

「実用性全振りの現代言語」

---

### ■ Java

* JVMでどこでも動く
* 大規模開発に強い（企業向け）
* ライブラリ・フレームワークが超豊富
* ガベージコレクションでメモリ管理不要

「安定性・実績・規模」

---

### ■ C#

* .NET環境で超強い（Windows系）
* 文法がモダンで書きやすい
* Unityでゲーム開発に強い
* 非同期処理（async/await）が優秀

「Javaの進化版みたいな立ち位置」

---

## スクリプト・柔軟系

### ■ Python

* 圧倒的に書きやすい
* AI・機械学習ライブラリ最強
* 試作が爆速
* 読みやすいコード

「開発速度ならトップ」

---

### ■ JavaScript

* ブラウザで動く唯一の言語
* フロントエンド必須
* Node.jsでサーバーも可
* エコシステムが巨大

「Webでは絶対王者」

---

### ■ Ruby

* 文法が自然言語に近い
* RailsでWeb開発が高速
* コードが美しく書ける

「気持ちよく書ける言語」

---

### ■ VB.NET

* シンプルで初心者向け
* GUIアプリが作りやすい
* .NETの恩恵を受けられる

「とっつきやすさ重視」

---

##  特殊・思想系

### ■ Haskell

* 純粋関数型（副作用なし）
* 数学的に正しいコードが書ける
* 並行処理・抽象化が非常に強力
* コードの美しさが際立つ

「理論・美しさ・安全性」

---

## まとめ

* **速度重視** → C / C++ / Rust / Fortran
* **実用バランス** → Go / Java / C#
* **開発速度・柔軟性** → Python / JS / Ruby
* **思想系** → Haskell
